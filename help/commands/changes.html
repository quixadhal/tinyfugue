/changes

Usage:

/CHANGES [version]

List the changes in a version of TinyFugue; if omitted, version defaults to the current version. Version can be a full version name (e.g., "3.5 beta 1") or just the major and minor numbers (e.g., "3.5"). The information is kept in the file %TFLIBDIR/CHANGES.

A list of changes in the latest version of tf can be found at http://tf.tcp.com/~hawkeye/tf/CHANGES.

See: /version
